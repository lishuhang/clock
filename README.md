**核心特性：**

- Canvas 绘制矢量时钟，
- 时钟色彩 50% 透明度，自动检测背景亮度（亮底用深色、暗底用白色）
- 数字时钟位于表盘中心与6点钟之间，冒号每秒闪烁
- 时钟始终居中，自适应窗口大小并保持舒适边距

**音频同步：**

- 音乐根据当前秒数定位播放位置（确保与时钟节拍同步）
- 无缝循环，自动漂移校正（误差超过 0.2 秒时重新同步）

**设置面板（右上角齿轮）：**

- 可拖动
- 视频 URL 设置
- 音频 URL 设置
- 时区选择（本地/UTC/各时区）
- 音乐开关
- 自动保存到 localStorage

**部署说明：** 将 `clock.html`、`clock.mp3` 和你的视频文件（默认名 `cloud.mp4`）放在同一目录即可。支持本地测试和 Cloudflare Pages/GitHub Pages 部署。

Inter 字体通过 Google Fonts 加载（Light 300 用于设置界面，Extra Bold 800 用于数字时钟）。
